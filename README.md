# starter
A starter template using:
  - PostCss -Plugins:
     - import
     - mixins
     - autoprefixer
     - nested
     - simple vars
     - hexrgba
     
  - jQuery
  - Babel
  - Gulp
  - Webpack 2
 
# instruction
- Make sure Node and npm are installed on your system.
- you need to install Webpack globally: npm install webpack -g
- Run: "npm install" in terminal.
- Run: "gulp watch"
